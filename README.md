✨ 核心功能（由 AI 赋能）

AI Bookmarks 的核心在于其智能、无需手动干预的管理能力：

🧠 AI 智能分类与归档：

告别手动整理文件夹。用户只需点击“一键收藏”，Gemini 就会即时阅读网页内容，分析其语义上下文，并自动将其归档到最合适的文件夹层级中。

自动生成关键词标签和简洁摘要。

💬 语义化自然语言搜索：

忘了书签的准确名称？没关系。用户可以使用自然语言描述（例如：“上周保存的关于 Vue.js 性能优化的文章”），AI 就能精准定位并召回相关结果。
